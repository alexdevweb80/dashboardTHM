Les rooms ci-dessous m'ont permis de mettre en pratique les cours théoriques

# RECONNAISSANCE & OSINT

  → nmap          — scan de ports et détection de services

  → whois         — informations sur les domaines

  → dig / nslookup — résolution DNS, zones de transfert

  → Burp Suite    — interception HTTP, manipulation de requêtes

  → gobuster      — directory busting, fuzzing d'endpoints

  → Bash / PowerShell — automatisation, scripting système

  → Wireshark     — analyse de trames réseau

  → netcat (nc)   — connexions TCP/UDP, reverse shells

  → SQLMap        — injection SQL automatisée

  → CyberChef     — encodage, décodage, analyse de données

  Chaque room terminée = un vecteur d'attaque mieux compris,
